# App de Productividad con IA

## Descripción
App móvil y web que organiza tareas, eventos y objetivos con IA.

## Stack Técnico
- Frontend: React Native + Expo (móvil), Next.js (web)
- Backend: Node.js + Express
- Base de datos: PostgreSQL + Prisma
- IA: OpenAI API
- Auth: Clerk
- Hosting: Vercel (frontend) + Railway (backend)

## Modelo de Negocio
Freemium con límites en plan gratuito.
